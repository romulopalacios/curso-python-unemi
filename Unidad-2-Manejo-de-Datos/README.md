# Unidad 2: Manejo de Datos y Programación Intermedia

En esta unidad elevamos el nivel lógico para manejar colecciones complejas de datos, modularizar el código y conectarlo con archivos del sistema. Estos son los cimientos para construir arquitecturas backend eficientes.

### Archivos de Código
* `u2_tema_1_listas_tuplas_conjuntos_diccionarios.py`: Implementación y manipulación de estructuras de datos. Destaca el uso de diccionarios anidados para simular registros de bases de datos y *List Comprehensions* para filtrado eficiente.
* `u2_tema_2_funciones_modular.py`: Creación de bloques de código reutilizables mediante funciones con parámetros múltiples y valores por defecto, además de la importación de módulos nativos (`math`).
* `u2_tema_3_archivos_excepciones.py`: Apertura, lectura y escritura de archivos de texto (`.txt`) usando el manejador de contexto `with open()`, junto con el control de errores estructurales mediante `try/except`.
