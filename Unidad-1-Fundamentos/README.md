### Tema 1: Introducción y Arquitectura

**Conceptos clave:**
- ¿Qué es Python? Lenguaje interpretado vs compilado
- Arquitectura de Hardware: CPU, RAM, Memoria Secundaria
- Entorno de desarrollo: VS Code, Google Colab
- Variables y asignación de memoria

### Tema 2: Variables, Tipos de Datos y Operadores

**Conceptos clave:**
- Variables y nombres mnemónicos
- Tipos de datos: `int`, `float`, `str`, `bool`, `complex`
- Estructuras básicas: `list`, `tuple`, `dict`
- Operadores: aritméticos, comparación, lógicos
- Tipado dinámico en Python

### Tema 3: Estructuras Condicionales y Bucles

**Conceptos clave:**
- Control de flujo: ejecución secuencial vs condicional
- Condicionales simples (`if`), dobles (`if-else`) y múltiples (`elif`)
- Condicionales anidados y encadenados
- Bucles indefinidos (`while`) y definidos (`for`)
- Modificadores: `break` y `continue`
- Manejo básico de excepciones

**📄 Archivo de práctica:**  
[`u1_tema_3_estructuras_condicionales_y_bucles.py`](./u1_tema_3_estructuras_condicionales_y_bucles.py)

**Ejercicios incluidos:**
- Validación de edad con condicionales
- Sistema de calificaciones con `elif`
- Autenticación básica con condicionales anidados
- Iteraciones con `while` y `for`
- Uso de `break` y `continue`

## 💡 Conceptos Clave a Recordar

| Concepto | Descripción |
|----------|-------------|
| **Variable** | Espacio en RAM con un nombre asociado que almacena un valor |
| **Tipado Dinámico** | Python infiere el tipo automáticamente |
| **Inmutabilidad** | Las tuplas y strings no pueden modificarse una vez creados |
| **Evaluación en Cortocircuito** | `and` y `or` no evalúan todas las expresiones si no es necesario |
| **PEP 8** | Guía de estilo oficial de Python (nombres en snake_case) |
